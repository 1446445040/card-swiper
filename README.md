# 轮播小插件Swiper
一个组件，一共三百行的样子，比较容易看懂。然后主要文件是CardSwiper.vue

# 参数
```
images: Array              // 必须。图片链接数组
width: Number | String     // default: 500,    单位px
height: Number | String    // default: 250,    单位px
autoplay: Boolean | Number // default: 3000,   单位ms
```
